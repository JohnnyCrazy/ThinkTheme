
# ThinkTheme

My first theme using polybar and i3.

## Installation

### Archlinux / Manjaro

```sh
yaourt -S aur/polybar \
          aur/i3-gaps \
          extra/noto-fonts \
          aur/ttf-material-icons \
          aur/termsyn-font \
          aur/otf-fontawesome \
          aur/ttf-unifont \
          aur/i3lock-fancy-git \
	  community/ttf-fira-mono
	  extra/feh
```

then move the .config file in your home reload i3 it should works :)
